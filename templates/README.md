# 模板参考

这里是通用模板，供各编辑器初始化时参考。

## 目录结构

```
templates/
├── zh/                      # 中文模板
│   ├── file-header-ts.txt   # TypeScript 文件头注释
│   └── folder-readme.md     # 文件夹 README 模板
└── en/                      # 英文模板
    ├── file-header-ts.txt
    └── folder-readme.md
```

## 使用说明

这些模板是**参考**，AI 在执行初始化任务时会根据项目实际情况生成适配的版本。

如果需要其他语言的文件头注释模板（Go/Rust/Python），AI 会根据 TypeScript 模板的格式自动生成。
